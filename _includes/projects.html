OpenCarto Web GIS Framework

For the past year and a half I have been a lead front-end developer and product manager for the OpenCarto platform, a full stack framework developed at the National Renewable Energy Lab for creating feature-rich web GIS web applications to enable visualization and analysis of renewable energy data. In this role I did significant client-side development using Ember.js, Leaflet, and jQuery, in addition to analysis and data management work in PostgreSQL/PostGIS and Geoserver.

Boulder County Environmental Education Assessment

The BCEE 2014 Report Card was a first-of-its-kind effort to comprehensively understand the environmental education being received by students in Boulder County across schools, grade levels, and demographics. As part of this project, I performed some basic statistical analysis and data munging, created static visuals for the final report, and created an interactive web map for the Report's website.

Greening the Grid India Visualization

The Greening the Grid India Visualization is an interactive application for exploring the results of grid integrations studies conducted at NREL under the Greening the Grid program. The application visualizes one year of hourly time-series generation and power flow data under a variety of renewable penetration scenarios. It is built on a Backbone.js framework, and uses D3.js and TopoJSON to create the maps and charts. The application has an extremely simple backend, consisting of flat JSON files served directly to the client, with the majority of the data handling happening in the frontend using D3.js and underscore.js.
